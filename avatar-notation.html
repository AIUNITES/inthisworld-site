Avatar Default States

These values should be set on avatar spawn. All use the standard [Con:] activation levels. Level 0 (zero) is valid for genital/sphincter structures only and means fully relaxed / no tone.

SymbolDefaultRationale
Sph.IA+Involuntary resting tone — always present
Sph.EA+Voluntary resting closure
AV.Cush+Vascular cushions always provide baseline closure
PF.PC+Minimal pelvic floor resting tone
Corp.Cav / Corp.C0Fully relaxed unless explicitly set
Glan.P / Glan.C0No activation unless set

Implementation Note for Avatar Rigs

Pelvic floor and genital structures are driven by blend shapes or soft-body simulation, not skeletal rotation. They have no [Pos:] joint entries in the MNN spec — only [Con:] activation levels map to morph targets.

For Three.js (used in InThisWorld): use morphTargetInfluences on your avatar mesh geometry. Map each MNN symbol to a named blend shape key. Activation level + = ~0.3, ++ = ~0.65, +++ = 1.0, 0 = 0.0.

// Example Three.js morph target mapping
const MNN_MORPH_MAP = {
  'Sph.EA':    'sphincter_external_anal',
  'Sph.IA':    'sphincter_internal_anal',
  'PF.PC':     'pelvic_pubococcygeus',
  'Corp.Cav':  'penis_corpus_cavernosum',
  'Glan.C':    'clitoris_glans',
  'Vag.Or':    'vaginal_introitus',
  // ... etc
};

const LEVEL_VALUE = { '0': 0.0, '+': 0.3, '++': 0.65, '+++': 1.0 };

function applyMNN(mesh, symbol, level) {
  const key = MNN_MORPH_MAP[symbol];
  if (!key) return;
  const idx = mesh.morphTargetDictionary[key];
  if (idx !== undefined) mesh.morphTargetInfluences[idx] = LEVEL_VALUE[level] || 0;
}
